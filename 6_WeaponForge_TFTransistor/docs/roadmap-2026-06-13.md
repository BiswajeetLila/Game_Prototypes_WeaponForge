## Decision gates (between phases)

### Gate 3 → 4: Spec LOCK
- User reads REVIEW-3 spec end-to-end
- All 26 locked decisions in spec §21 confirmed
- All 8 Phase 4 scope sections in slice doc confirmed
- → LOCKED, cut `weaponforge-tftransistor/vertical-slice` branch

### Gate 4 → 4.5: Slice freeze
- Phase 4 build sequence §4 complete (18 TDD steps)
- 90+ tests green
- ≥30 FPS on baseline phone
- AUTOSHOT capture of 6 key beats
- → tag `6_WF_TFT/phase4-slice-frozen-YYYY-MM-DD`, begin playtest sessions

### Gate 4.5 → 5: Feel-gate (THE critical gate)
3-objective evaluation per Phase 4 scope doc §5:
1. **Forge feel** — testers articulate a build decision they made
2. **Magicka chain dopamine** — testers experience cross-hero reaction chains
3. **FTUE retention** — testers reach stage 3 first-Steam moment alive

**All 3 pass:**
- Greenlight Phase 5 full rewrite
- Lock pitch positioning (Backpack Hero / Slice & Dice / Brotato cohort vs Wittle/AFK)
- Begin Wittle-meta-progression spec doc (separate)
- Begin monetization design discussion

**Objective 1 passes, 2 fails (forge OK but combat thin):**
- Activate 3-card in-combat module (function spec §20)
- Re-test in Phase 4.5 patch
- Decide Phase 5 entry after re-test

**Objective 3 fails (FTUE retention):**
- Redesign FTUE pacing in Phase 4.5
- May require structural change (stage length, forced-shop curriculum)
- Re-test before Phase 5

**≥2 fail:**
- Consider fork B (smaller-scope pivot) or fork C (return to 2_WC polish)
- Hold Phase 5; evaluate forks
- User makes final call (no sunk-cost commitment)

### Gate 5 → 5+: Phase 5 freeze
- Full 12 Functions + 15 reactions live
- Real boss AI per stage 5 W3
- Tier system T1-T5 active
- Multi-world progression (heroes persist; Functions reset per world)
- 210+ tests green
- → cut Wittle-meta spec branch

### Gate 5+ → 5++: Meta-layer ready
- Hero levels + skill trees + equipment + talents + stars implemented
- Idle income + daily challenges + season pass structure stubbed
- → monetization spec branch

### Gate 5++ → 6: Soft launch
- Monetization model live (IAP / battle pass / cosmetic — TBD by playtest)
- Save-cloud sync
- Localization (EN + 1-2 others)
- Analytics + crash reporting
- → soft launch on closed beta channels

### Gate 6 → 7: Open launch
- Soft launch D1/D7/D30 retention ≥ cohort baseline
- Crash rate <1%
- Average session length ≥ 4 minutes (target = 1 world)
- → public launch
