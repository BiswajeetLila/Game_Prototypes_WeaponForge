# WeaponCraft — Roadmap (P0 → full game)

**Direction locked 2026-06-11.** Full design + rationale: [`superpowers/specs/2026-06-11-hero-squad-meta-design.md`](superpowers/specs/2026-06-11-hero-squad-meta-design.md).

Sequencing view of the "crafting-core run + persistent hero-squad gacha meta" build. One layer at a time — each phase ships and tests independently (no "red-dot hell").

---

## Now: P0 — first prototype (on top of the current build)

**Validates:** *"Do players return for session 2–3 when heroes persist + level?"* — cheapest test of the whole direction, **no gacha yet.**

- Persistence/save layer (current build has none).
- HOME screen: roster + per-hero Level + Form Squad (≤3) + Battle.
- Pre-run squad-select; **removes** the in-run scripted unlock (Elara W3 / Vex W6).
- Hero Level — persistent XP from runs → base stats.
- Result/rewards screen (visible reward→return link).
- Juice + **audio** pass (audio is currently unbuilt).

**Non-goals:** gacha, Star-up, Mastery, new heroes, passive kits, synergy, permanent weapons, stage telegraph, expeditions, IAP, stamina.

---

## P1 — collection + counter-build (first monetizable shape)
- Gacha pull + pity (50 soft / 100 hard, starting) + 2–3 new heroes.
- Star-up via duplicate shards.
- Gems + Hero Shards economy; **stamina gate** (required before persistent currency).
- **Stage-affinity telegraph** — half-telegraphed stage threat → craft a "set" for the 15-wave stage.

## P2 — narrative depth + crafting permanence
- Hero Mastery / bond track + dialogue unlocks.
- Portrait evolution (visible hero growth at Mastery milestones).
- **Permanent knowledge** — craft a recipe N times → unlock it as a known blueprint (codex → build-craft).

## P3 — full hero richness
- Passive kits (one run-altering passive per hero).
- **Hero signature weapons** (Mastery-gated, hero-locked, craft-and-keep).
- Squad synergy (light element/class team buffs).

---

## Roadmap / live-ops (post-core)
- **Expeditions / discovery** — send benched heroes out → find recipes/runes/items + level them; new currency; solves roster bloat (Yakuza-0-Majima-flavored).
- **Seasons** — resettable monthly track; preserves permanent power, re-levels returners (AFK-Journey pattern).
- **Difficulty tiers** — Elite / Nightmare wave replays for better material yields.
- **Battle pass**, **cosmetics** (hero skins, weapon glow, ult VFX).
- Later: PvP arena, guilds (Year 2 — opportunistic).

---

## Launch shape (target)
- **Roster:** 3 free (Bran/Elara/Vex) + 6–9 pullable; +1 hero/month live-ops. (Watch the small-roster D30 content-cliff.)
- **Currencies:** Gold · Gems · Hero Shards · Hero XP (4 — minimal by design; a 5th only with expeditions).

## Strategic notes
- **Crafting stays the star** (≥60% run power). Heroes are the meta wrapper, never a replacement for craft decisions — or we dilute the moat and collide with `5_WeaponForge`.
- **Architect persistence cleanly in P0.** Save schema should anticipate gacha/Star/Mastery fields even if unused in P0 — cheaper than re-architecting later.
- **Lock pull rates + pity before any public build.** Changing them post-launch causes backlash.
- **Content cadence > content depth** for casual-mobile retention.
