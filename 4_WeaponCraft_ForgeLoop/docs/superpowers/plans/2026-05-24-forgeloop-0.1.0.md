# ForgeLoop 0.1.0 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship the first playable build of the ForgeLoop variant — an 8-cell Loop-Hero-style forge ring with 4 modal minigames, a 4-second forge reveal climax, and a weapon that persists across waves and visibly arms the hero during combat.

**Architecture:** Single-file HTML/JS prototype forked from `2_WeaponCraft_Base/Prototype/dist/BASE-A1_0.1.9.html`. Combat layer, hero card, and parchment-brass theme are reused as-is. The TFT shop, 3-slot anvil, recipe codex, discovery overlay, and 2 of 3 heroes are stripped. A new ring UI + tile tray + drag-drop + minigame framework + forge reveal sequence replaces the crafting layer. Output is a single file: `Prototype/dist/FORGELOOP_0.1.0.html`. All JS lives inline. No build step, no bundler, no test framework — verification happens by loading the file in a browser and checking DOM state + console event log against the schema.

**Tech Stack:** Vanilla HTML5 + CSS3 + JS (ES2022). SVG for ring + weapon sprite composite. CSS transitions + `requestAnimationFrame` for ingot travel. Pointer events for drag-drop (touch + mouse unified). No external dependencies. Verification via the gstack `/browse` skill (headless Chromium) or any modern desktop browser.

## Testing approach (read first)

This is a single-file HTML prototype with no automated test framework. The base build never had one. Adding pytest/jest here would create a build step we don't want.

Instead, each task follows this loop:

1. **Define the expected console event(s)** for the task — the schema is in the design spec § "Console event schema". Each task lists which events its step should emit.
2. **Implement the minimal code.**
3. **Verify in a browser** — either:
   - **Manual:** open `FORGELOOP_0.1.0.html` in Chrome/Firefox, open DevTools console, perform the action, confirm the expected event(s) appear and the DOM state matches the description.
   - **Automated:** use the gstack `/browse` skill — `browse open file:///…/FORGELOOP_0.1.0.html`, perform actions via `browse click`/`browse type`, dump console via `browse console`, assert against expected event shape.
4. **Commit** with a focused message.

Console events double as our test assertions: if you instrumented correctly, the event log fully describes what the user did and what the system computed. The `window.__forgeLoopStats()` helper (built in Task 14) snapshots current state for ad-hoc inspection.

Some tasks also add a `console.assert(...)` invariant line directly in the code (e.g., "ring has exactly 8 cells", "stats are non-negative"). Treat these as in-code unit tests — if `assert` fails in DevTools, the build is broken.

```javascript
// Cell 0 = top (Smelter). Cells go clockwise: 0,1,2,3 down right side; 4 = bottom (Inspector); 5,6,7 up left side.
// Polar layout: angle = -90deg + (i * 45deg) so cell 0 is at 12 o'clock.
const RING_RADIUS = 130;  // px, from center of #ring-stage
const CELL_SIZE = 64;     // px, hex cell width

function cellAngleDeg(i) {
    return -90 + i * (360 / RING_CELLS);
}

function cellCenterPx(i) {
    const a = cellAngleDeg(i) * Math.PI / 180;
    return { x: RING_RADIUS * Math.cos(a), y: RING_RADIUS * Math.sin(a) };
}

console.assert(RING_CELLS === 8, 'ring must be 8 cells');
console.assert(Math.abs(cellCenterPx(0).y + RING_RADIUS) < 0.001, 'cell 0 must be at top');
console.assert(Math.abs(cellCenterPx(4).y - RING_RADIUS) < 0.001, 'cell 4 must be at bottom');
```

- [ ] **Step 2: Implement drag-drop in `§ DRAG_DROP`**

Use pointer events for touch+mouse unification.

```javascript
let dragState = null;  // { stationId, trayIdx, ghostEl, originTile }

function attachDrag(tileEl) {
    tileEl.addEventListener('pointerdown', onDragStart);
}

function onDragStart(e) {
    if (state.lapInProgress) return;  // no edits mid-lap
    const tile = e.currentTarget;
    tile.setPointerCapture(e.pointerId);
    const stationId = tile.dataset.stationId;
    const trayIdx = parseInt(tile.dataset.trayIdx, 10);
    const ghost = document.createElement('div');
    ghost.className = 'tile-ghost';
    ghost.textContent = stationIcon(stationId);
    ghost.style.left = e.clientX + 'px';
    ghost.style.top = e.clientY + 'px';
    document.body.appendChild(ghost);
    tile.classList.add('dragging');
    dragState = { stationId, trayIdx, ghostEl: ghost, originTile: tile, pointerId: e.pointerId };
    tile.addEventListener('pointermove', onDragMove);
    tile.addEventListener('pointerup', onDragEnd);
    tile.addEventListener('pointercancel', onDragEnd);
}

function onDragMove(e) {
    if (!dragState) return;
    dragState.ghostEl.style.left = e.clientX + 'px';
    dragState.ghostEl.style.top = e.clientY + 'px';
    // Highlight drop target.
    document.querySelectorAll('.ring-cell.drop-target').forEach(c => c.classList.remove('drop-target'));
    const cell = cellUnderPointer(e.clientX, e.clientY);
    if (cell && canDropOnCell(cell)) cell.classList.add('drop-target');
}

function onDragEnd(e) {
    if (!dragState) return;
    const cell = cellUnderPointer(e.clientX, e.clientY);
    const dropped = cell && canDropOnCell(cell);
    if (dropped) {
        const cellIndex = parseInt(cell.dataset.cellIndex, 10);
        placeStation(dragState.stationId, dragState.trayIdx, cellIndex);
    }
    dragState.ghostEl.remove();
    dragState.originTile.classList.remove('dragging');
    document.querySelectorAll('.ring-cell.drop-target').forEach(c => c.classList.remove('drop-target'));
    dragState = null;
}

function cellUnderPointer(x, y) {
    const el = document.elementFromPoint(x, y);
    return el && el.classList.contains('ring-cell') ? el : null;
}

function canDropOnCell(cell) {
    if (cell.classList.contains('fixed')) return false;
    return cell.classList.contains('empty');
}

function placeStation(stationId, trayIdx, cellIndex) {
    state.ring[cellIndex] = stationId;
    state.tray.splice(trayIdx, 1);
    emit('station_placed', { wave: state.wave, cellIndex, stationId, fromTray: true });
    snapFitFeedback(cellIndex);
    renderAll();
    refreshStartLapButton();
}

function snapFitFeedback(cellIndex) {
    // S7 Lego snap: scale-pulse + click SFX (SFX deferred — visual only in 0.1.0).
    setTimeout(() => {
        const cell = document.querySelector(`.ring-cell[data-cell-index="${cellIndex}"]`);
        if (!cell) return;
        cell.animate(
            [{ transform: cell.style.transform + ' scale(1.0)' },
             { transform: cell.style.transform + ' scale(1.18)' },
             { transform: cell.style.transform + ' scale(1.0)' }],
            { duration: 220, easing: 'cubic-bezier(0.34, 1.56, 0.64, 1)' }
        );
    }, 10);
```

```javascript
function accumulateLapStats(stationId, result) {
    const s = state.lapStats;
    s.atk += result.stats.atk || 0;
    s.crit += result.stats.crit || 0;
```

```javascript
    bodyEl.innerHTML = `
        <div class="engraver-palette">
            <button class="glyph-pick" data-glyph="fire">🔥 Fire</button>
            <button class="glyph-pick" data-glyph="ice">❄️ Ice</button>
            <button class="glyph-pick" data-glyph="pierce">✶ Pierce</button>
        </div>
        <canvas id="engraver-canvas" width="200" height="200"></canvas>
        <div id="engraver-coverage">Pick a glyph above</div>
    `;
    hintEl.textContent = 'Pick a glyph, then trace the dotted path in one stroke';
    let glyph = null;
    let path = [];
    let dragging = false;
    const CANVAS = document.getElementById('engraver-canvas');
    const ctx = CANVAS.getContext('2d');
    const COV_EL = document.getElementById('engraver-coverage');

    function drawGlyph() {
        ctx.clearRect(0, 0, 200, 200);
        if (!glyph) return;
        // Dotted target path.
        ctx.setLineDash([4, 6]);
        ctx.strokeStyle = glyph.color;
        ctx.lineWidth = 2;
        ctx.beginPath();
        glyph.nodes.forEach((n, i) => {
            const x = n[0] * 200, y = n[1] * 200;
            if (i === 0) ctx.moveTo(x, y);
            else ctx.lineTo(x, y);
        });
        ctx.stroke();
        // Trace path (solid).
        ctx.setLineDash([]);
        ctx.strokeStyle = '#fff';
        ctx.lineWidth = 3;
        ctx.beginPath();
        path.forEach((p, i) => i === 0 ? ctx.moveTo(p.x, p.y) : ctx.lineTo(p.x, p.y));
        ctx.stroke();
    }

    function coverage() {
        if (!glyph) return 0;
        // For each segment between consecutive glyph nodes, sample N points and count how many
        // lie within 18px of the player's traced polyline.
        const RADIUS = 18;
        const SAMPLES_PER_SEG = 12;
        let totalSamples = 0, hits = 0;
        for (let i = 0; i < glyph.nodes.length - 1; i++) {
            const a = glyph.nodes[i], b = glyph.nodes[i + 1];
            for (let s = 0; s < SAMPLES_PER_SEG; s++) {
                const t = s / (SAMPLES_PER_SEG - 1);
                const x = (a[0] + (b[0] - a[0]) * t) * 200;
                const y = (a[1] + (b[1] - a[1]) * t) * 200;
                totalSamples++;
                if (path.some(p => Math.hypot(p.x - x, p.y - y) <= RADIUS)) hits++;
            }
        }
        return totalSamples ? hits / totalSamples : 0;
    }

    bodyEl.querySelectorAll('.glyph-pick').forEach(btn => {
        btn.addEventListener('click', () => {
            glyph = GLYPHS[btn.dataset.glyph];
            path = [];
            COV_EL.textContent = 'Trace now (one stroke)';
            drawGlyph();
        });
    });

    CANVAS.addEventListener('pointerdown', (e) => {
        if (!glyph) return;
        dragging = true; path = [];
        const r = CANVAS.getBoundingClientRect();
        path.push({ x: e.clientX - r.left, y: e.clientY - r.top });
        CANVAS.setPointerCapture(e.pointerId);
    });
    CANVAS.addEventListener('pointermove', (e) => {
        if (!dragging) return;
        const r = CANVAS.getBoundingClientRect();
        path.push({ x: e.clientX - r.left, y: e.clientY - r.top });
        drawGlyph();
        const c = coverage();
        COV_EL.textContent = `Coverage: ${(c * 100).toFixed(0)}%`;
    });
    CANVAS.addEventListener('pointerup', () => {
        if (!dragging) return;
        dragging = false;
        const c = coverage();
        logMinigameInput('engraver', 'trace', { glyphId: glyph.id, pathLen: path.length, coverage: c });
        const perf = c >= 0.85 ? 'perfect' : (c >= ENGRAVER_TRACE_PASS_PCT ? 'good' : 'soft_fail');
        const element = (c >= ENGRAVER_TRACE_PASS_PCT) ? glyph.id : null;
        resolve({ score: Math.round(c * 100), perf, stats: { element, glyphIcon: element ? glyph.icon : null, glyphColor: element ? glyph.color : null } });
    });
};
```

```javascript
// === SECTION: WEAPON ===

function buildWeaponFromLap(lapStats) {
    const quality = qualityAdjective(lapStats.perfBuckets);
    const elementAdj = elementAdjective(lapStats.element);
    const baseType = 'Shortsword';
    const name = [quality, elementAdj, baseType].filter(Boolean).join(' ');
    return {
        name,
        atk: lapStats.atk,
        crit: Math.min(100, lapStats.crit),
        dur: lapStats.dur,
        element: lapStats.element,
        elementIcon: lapStats.element ? GLYPHS[lapStats.element].icon : null,
        elementColor: lapStats.element ? GLYPHS[lapStats.element].color : null,
        layers: layerPlan(lapStats),
    };
}

function qualityAdjective(perfBuckets) {
    if (perfBuckets.length === 0) return 'Plain';
    const hasSoft = perfBuckets.some(b => b.perf === 'soft_fail');
    const allPerfect = perfBuckets.every(b => b.perf === 'perfect');
    const allGoodOrBetter = perfBuckets.every(b => b.perf !== 'soft_fail');
    if (allPerfect) return 'Masterwork';
    if (allGoodOrBetter && !hasSoft) return 'Tempered';
    if (hasSoft && perfBuckets.some(b => b.perf === 'perfect' || b.perf === 'good')) return 'Honed';
    return 'Crude';
}

function elementAdjective(element) {
    return { fire: 'Flame-Etched', ice: 'Frost-Etched', pierce: 'Pierce-Etched' }[element] || '';
}

function layerPlan(lapStats) {
    // Composite layers for sprite: base blade, tint, glyph overlay.
    return {
        baseTint: lapStats.dur >= 4 ? 'cool' : 'warm',  // good quench → bluer steel
        glyph: lapStats.element,
        sheen: lapStats.perfBuckets.length >= 3 ? 'high' : 'low',
    };
}
```
