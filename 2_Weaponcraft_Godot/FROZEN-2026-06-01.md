## What this build plays (the frozen state)

**Stage D prototype** — the 15-wave vertical slice:

- **3 heroes:** Bran (warrior), Elara (mage), Vex (rogue). Unlock at W1 / W3 / W6.
- **15 waves**, bosses at **W5 / W10 / W15**.
- **Reforge-&-Retry** on boss wipe.
- **Legacy 3-socket forge** (head / hilt / rune shop + merge — the slow loop; *not* the
  new Forge Wheel / weapon-gacha, which is design-only at this point).

**NOT in this build (design-only, never started here):** Forge Wheel slot-machine
pulls, weapon-gacha, Forge Draft 3-card, Catalyst compounds, Hot Paladin cinematic,
hero missions, portrait evolution. `WeaponData` exists in code but is **not** wired
into gameplay in this frozen build.

**Tests at freeze:** 144/144 core green (TestCombat / TestRecipes / TestShop /
TestMerge / TestUi) + 10/10 `TestWeaponData`.
