## Class roles (high-level)

- **Warrior**: tank, melee damage, high HP. Universal-tag friendly.
- **Mage**: ranged caster, elemental scaling. High affinity for Fire/Ice/Lightning/Shadow runes.
- **Rogue**: high crit, low HP, fast attacks. High affinity for Bleed/Pierce/Poison.
- **Ranger**: ranged DPS, multi-target. Affinity for Pierce/Lightning.
- **Paladin**: support tank, party-heal flavor. Affinity for Holy.
- **Druid**: hybrid melee/caster, shapeshifter. Affinity for Nature (new element TBD).
- **Necromancer**: DoT specialist, summons. Affinity for Shadow/Poison.
- **Berserker**: glass-cannon melee. Affinity for Bleed/Fire.
- **Monk**: fast-attack hybrid. Affinity for Lightning/Holy.

## Star-Up tiers

Each hero has 5 star tiers (★1 → ★5):
- **★1**: base stats + base ultimate.
- **★2**: +5% all stats + ult cosmetic upgrade tier 1.
- **★3**: +10% all stats + minor passive ("on stage start: +1 round currency").
- **★4**: +15% all stats + ult cosmetic upgrade tier 2 (different VFX).
- **★5**: +20% all stats + signature passive ("ultimate cost -50%").

Damage scaling is gentle: ★1 vs ★5 power gap target is **~1.5×**, not 5×. F2P-at-★1 stays competitive in main campaign.
