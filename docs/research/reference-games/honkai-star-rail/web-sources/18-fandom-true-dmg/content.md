## True DMG

### Type

Combat
Non-Type DMG that is not affected by any effects. This DMG is not considered as using 1 attack.
**True DMG** is a type of damage dealt through certain abilities. True DMG is *not* considered an attack. It is also *not* modified by other multipliers during damage calculation.
