# Season

**Type:** Game Systems

**Seasons** are major, self-contained storylines in AFK Journey which bring the player to new regions of Esperia. Each season has its own unique mechanics, rewards, and progression lines that reset with each new season.

Excluding the starter story, only one live season is available at a time. Past seasons can be accessed through Peaks of Time, with some restrictions and changes to gameplay.

## Seasonal Progression

Each new season features new progression lines that are unique for each season. These seasonal progression systems are separate from the **permanent progression systems** featured in the Starter Story.

**Seasonal progression systems** function in tandem with the existing permanent system. Season progression comprises of new systems such as: seasonal resonance level, seasonal class equipment, seasonal artifacts, and potential new seasonal mechanics and items.

Seasonal progression is tailored to emphasize the strategic fun of progressing through new seasonal content. The design philosophy is built upon three key principles: to preserve the value of permanent progression systems, to create an easy-to-manage seasonal progression system, and to lower the entry barrier for returning newcomers.

| Progression Type | Description | Examples |
|---|---|---|
| Permanent Progression System | The foundations of a player's power. These systems are effective across all seasons, and have the greatest impact on the exploration of the main story quests and most game modes. | Starter Story artifacts, resonance levels, class equipment, hero tiers, hero-exclusive equipment, etc. |
| Seasonal Progression System | Progression systems that are developed by playing seasonal content. They are only effective in the season they belong to. | Seasonal resonance level, seasonal class equipment, seasonal artifacts, and potential new seasonal mechanics and items. |

All resources needed for the seasonal progression systems are available for free.

## Season Availability

To ensure a consistent experience for all players on a server, the new season will be available when the server has been active for a minimum set number of days. The earliest servers will gain access to the new season on launch day.

In order to make sure everyone knows when the season starts on their server, a countdown event banner is available in-game whenever a new season is approaching. The countdown will start 14 days before the season starts, adjusted based on when each server was opened. Typically, a season runs for 4 months and ends simultaneously for all servers running it, followed by a synchronized transition into the next season.

Unlocking seasonal content requires reaching Resonance Level 240 and completing the quest Ryeham Tales. Various mechanics are in place to help players reach these requirements before a new season launches.

These requirements ensure that players start a season on a level playing field, preventing significant progression and reward gaps.

## Seasonal Game Modes and Events

Each season features seasonal game modes and limited-time, seasonal events, that are only accessible during the current, live season. Once a season is over, many of these game modes and events will no longer be accessible, as they will be replaced with new game modes and events in the new season.

## Season Settlement

When a season concludes, all seasonal hero upgrades, progress, rankings, and rewards in seasonal modes will be reset. Servers will join a new District when the new season begins. If a server has few active players, it may also receive a new Allied Server when a season begins.

At the beginning of the new season, players receive rewards based on their progress with Season Resonance Level, Season Equipment, Season Artifacts, and Season Magic Charms from the previous season. Players can view details on how rewards are calculated and what they can earn before the new season kicks off via the Season Handbook.
