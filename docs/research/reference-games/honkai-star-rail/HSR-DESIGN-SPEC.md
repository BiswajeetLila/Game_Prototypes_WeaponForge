## 2. Core Game Loop

### Session-Level Loop (What a Player Actually Does)

[ASSUMED] A typical daily session runs 15–30 minutes for a committed player, structured as follows:

1. **Log in, collect login bonuses and Trailblaze Power (stamina) accumulation.** [GENUINE SOURCE] Reddit: "do dailies without needing to use my laptop. I use low settings tho."
2. **Complete 4 Daily Missions** (usually simple tasks: complete one battle, use one Skill Point, etc.) for ~60 Stellar Jade.
3. **Spend Trailblaze Power** (max 240, regenerates at 1/6 minutes) on Calyxes for character XP materials, Light Cone XP, Credits, or Relic farming domains.
4. **Advance main story quests or event missions** when available.
5. **Refresh endgame content** (Memory of Chaos, Pure Fiction, or Anomaly Arbitration resets bi-weekly) — optimizing team composition for the current rotation.
6. **Pull on active banners** when Stellar Jade thresholds are met.

[ASSUMED] The stamina system (Trailblaze Power) caps daily meaningful farming time at roughly 20–30 minutes of active play once the player is established. Story content extends sessions significantly when new patches drop.

### Macro Loop (The Strategic Arc)

The macro loop drives all monetization: [ASSUMED]

1. **Acquire character** — via limited banner gacha pull (anticipation → pull dopamine → reveal).
2. **Build character** — farm Ascension materials (Calyxes), level Traces, farm Relics.
3. **Clear endgame content** — attempt Memory of Chaos / Anomaly Arbitration with the new character.
4. **Hit a wall** — either a score ceiling in competitive endgame, or the current banner character proves to be the "solution" to a new enemy mechanic.
5. **Want the next character** — either to improve existing team, address a new mechanic, or respond to narrative hooks from the story.
6. **Save or spend Stellar Jade** — return to step 1.

[GENUINE SOURCE] A Play Store reviewer describes this loop from the negative perspective: "You bring the new character to the new cycle of whatever game mode it is, and then you just instantly wipe the entire thing." [GENUINE SOURCE] Video 02 (Braxophone) adds: "They log in, they pull the character, they build a character, they go slam their head against Endgame, and then they're done."

### Resource Flows

[ASSUMED based on KQM Beginner Guide + Video 03 structure]:

- **Trailblaze Power** → spent on Calyxes (XP/Credits/mats), Stagnant Shadows (Ascension mats), Echo of War (weekly boss mats), Cavern of Corrosion (Relics).
- **Stellar Jade** → converted to Passes → spent on Warps (gacha).
- **Skill Points** → generated by Basic ATKs, consumed by Skills in combat — a shared team resource.
- **Energy** → generated through damage dealt/received → spent on Ultimates in combat.
- **Credits** → currency for leveling, Trace upgrades, purchasing certain shop items.

### Traces (Character Upgrades)

[GENUINE SOURCE] KQM Beginner Guide: "Every character also has a number of Traces. Traces can be leveled to increase the potency of a character's abilities. - **Major Traces (A2/A4/A6):** Unique passive effects unlocked at Ascension 2, 4, and 6. Often define the character's optimal play pattern. Example: [GENUINE SOURCE] Ruan Mei A6: "In battle, for every 10% of Ruan Mei's Break Effect that exceeds 120%, her Skill additionally increases allies' DMG by 6%, up to a maximum of 36%."

- **Priority varies by character:** [GENUINE SOURCE] KQM Feixiao: "Ultimate >> Talent > Skill — Feixiao should avoid using her Basic ATK, so it can remain unleveled."

## 4. Combat System

### Turn Order / SPD / Action Value (AV)

[GENUINE SOURCE] KQM Beginner Guide: "Speed can affect initial turn order as well as how often your character goes. It is visually represented in-game through Action Value (AV), the numbers provided in the black boxes. Characters are arranged by lowest to highest AV, and the lowest AV character goes first."

Higher SPD = lower AV = acts more frequently. AV is hidden by default (requires enabling in settings). [GENUINE SOURCE] KQM: "Almost every action in Star Rail follows turn order, with one important exception. Ultimates can be cast at any time; they immediately interrupt the action queue after the current attack is completed."

**Action Advance:** [GENUINE SOURCE] Reduces a character's waiting interval before the next action. "100% Action Advance" means a character acts immediately. This is a core team-building consideration — Harmony characters like Bronya or Robin grant Action Advance to allies.

**Cycle counting in endgame:** [GENUINE SOURCE] Reddit MoC discussion: "Cycle 0 is 150 AV for each wave, which means 300 AV in total for two waves if utilized fully." Competitive players measure clear quality in "cycles used" and the number of limited 5-star characters required ("cost") for a given cycle count.

### Skill Types

[GENUINE SOURCE] KQM Beginner Guide: "Every character has a distinct set of five abilities: a Basic ATK, Skill, Ultimate, Talent, and Technique."

| Skill | Resource | Combat Use |
|---|---|---|
| Basic ATK | Generates 1 SP | Standard damage; types: Single Target, Blast, AoE |
| Skill | Consumes 1 SP | Core ability; cannot use at 0 SP |
| Ultimate | Requires full Energy | Powerful; can interrupt queue at any time |
| Talent | Passive | Always-active effect defining role |
| Technique | Technique Points (out of combat) | Pre-battle buffs or initiating combat with effects |

**Skill Point economy:** [GENUINE SOURCE] KQM: "You start each battle with 3 Skill Points. Using a Skill consumes Skill Points. Using a Basic ATK generates one Skill Point. You can hold up to 5 Skill Points, and Skill Points are shared across all characters within the team." Characters are labeled SP positive, SP neutral, or SP negative in team building discussions.

### Toughness Bar and Weakness Break

[GENUINE SOURCE] KQM: "Fully depleting an enemy's Toughness bar results in a Weakness Break, which delays the enemy's turn, deals extra damage, and applies a special effect based on the Element of the hit that caused the break." When Weakness Broken: "their resistances will drop and they will take more damage from any type."

**Silver Wolf Weakness implant:** [GENUINE SOURCE] KQM Silver Wolf guide: "There is a 75%/85%/87% base chance to add 1 Weakness of an on-field ally's Type to the target enemy. This also reduces the enemy's DMG RES to that Weakness Type by 20%." This mechanic enables mono-element team compositions that would otherwise lack Weakness coverage.

### Damage Formula

[GENUINE SOURCE] KQM Fu Xuan guide — Incoming DMG formula:

```
Total DMG Taken = Incoming DMG × Incoming DMG Multiplier
Incoming DMG Multiplier = DEF Multiplier × Type RES × DMG Reduction × Vulnerability
DEF Multiplier = 1 − (DEF / (DEF + 200 + 10 × Attacker Level))
```

**Key formula implications:**

- **DEF Reduction** (debuff on enemy) and **DEF Ignore** (buff on attacker) both work within the DEF Multiplier portion of the formula. [GENUINE SOURCE] KQM: "DEF Reduction can't go beyond 100%."
- **Vulnerability** is a separate multiplicative layer: [GENUINE SOURCE] KQM Topaz guide: "Proof of Debt is part of the Vulnerability multiplier in the damage formula. Since Vulnerability multipliers are rare and Proof of Debt is such a large multiplier, this makes it extremely strong."
- **Silver Wolf DEF Reduction value:** [GENUINE SOURCE] KQM at Level 80 characters vs Level 95 enemies — Silver Wolf Ultimate DEF Reduction alone: "+31.7% damage (no Weakness implant), +44.9% (Weakness already present), +81.1% (fresh Weakness implant with 20% RES reduction)."

### Endgame Modes

**Memory of Chaos (MoC):** [GENUINE SOURCE] KQM + Reddit: Rotating bi-weekly endgame with two sides requiring separate teams of 4. Stars awarded based on cycles used. [GENUINE SOURCE] Reddit: "11 million HP in 3.7" on MoC boss. Competitive players chase "0-cycle" clears (defeat all enemies within 150 AV per wave). [GENUINE SOURCE] Reddit: "Cycle 0 is 150 AV for each wave."

**Pure Fiction (PF):** [GENUINE SOURCE] KQM / Reddit: AoE-focused mode where "single-target attacks can be inefficient in scenarios with many enemies." Characters like The Herta "dominate with ease" while Feixiao (single-target focus) "struggles." Score-based.

**Anomaly Arbitration (AA):** [GENUINE SOURCE] Video 02 (Braxophone): "HSR's Stigian Onslaught equivalent — it's meant to be a game mode to push your limits." Structure: Three "Knight" encounters followed by "King in Check" boss. Star rewards based on cycle thresholds (6/4/2 cycles) and survival conditions. Enemy traits include Resilient Armor, Inversion, Energy Absorption, and Sunder. Specific team archetypes required per encounter — Break team, DoT team, and regular DPS. [GENUINE SOURCE] Video 02: "If you don't have a super break team, this is marginally harder. If you don't have a DOT team, good luck getting three stars." [GENUINE SOURCE] Reddit: "AA is straight up whale content with only teams that can 0 cycle it is e2s1 Phainon, Evernight, and Castorice."

**Simulated Universe / Divergent Universe:** [GENUINE SOURCE] KQM: Roguelite mode with Blessings that modify gameplay. Source of Star Rail Passes. Some characters (Herta, Qingque) are obtainable here.

### Traces Priority

[GENUINE SOURCE] Video 03: "Always do these first. They are what empower your character heavily." Trace priority varies by character: [GENUINE SOURCE] KQM Feixiao: "Ultimate >> Talent > Skill." Major Traces (A2/A4/A6) are the highest-priority unlocks and require Ascension.

## 15. Design Lessons for WeaponCraft

### Lesson 1: Every Character Needs a Clearly Named Role Identity

**Evidence:** [GENUINE SOURCE] HSR's 9 Paths are defined not just by mechanics but by philosophy. "Those who follow the 'Nihility' Path admire laziness, exhaustion, and meaningless behavior." Each Path name communicates personality, not just function. The character's path defines how players talk about, compare, and collect them.

**Application for WeaponCraft:** Give each weapon or weapon component a distinct archetype identity beyond stats. A "Hunt-equivalent" weapon should be described in flavor terms ("relentless," "precise") as well as mechanical terms ("single-target burst"). Players who collect should feel they are expressing an identity through their weapon roster, not just optimizing numbers.

---

### Lesson 2: The Weakness Break System Creates Mandatory Team Diversity Without Forcing It

**Evidence:** [GENUINE SOURCE] KQM: "Fully depleting an enemy's Toughness bar results in a Weakness Break, which delays the enemy's turn, deals extra damage, and applies a special effect." Each element produces a categorically different break effect — some provide CC (Freeze, Imprisonment), others maximize damage (Bleed, Burn). No single element is universally best. [GENUINE SOURCE] Silver Wolf's Weakness implant can artificially add any weakness, enabling mono-element teams but at the cost of one team slot.

**Application for WeaponCraft:** Design a comparable "break" layer in weapon interactions. Enemy types could have material vulnerabilities (e.g., soft-metal golems resist slashing but break against blunt impacts), each producing a different effect on break (stagger, bleed, armor strip). This naturally incentivizes players to own diverse weapon types — not because the game forces it, but because different enemies create different optimal solutions. WeaponCraft's modular weapon system (hilt + blade + rune) could use material-type combinations to produce different break effects.

### Lesson 3: Team Composition Depth Drives Long-Term Retention Better Than Individual Power

The forge loop itself can have synergies — certain hammer + anvil + catalyst combinations produce emergent effects players discover and share.

### Lesson 4: The Bimodal Review Split Reveals the Exact Moment to Avoid Crossing

**Evidence:** [GENUINE SOURCE] The most-upvoted negative Play Store review (👍3,774) targets a specific patch (v3.1.0) and a specific mechanic (global passives): "they've introduced global passives and with more to come... if you don't have these chars, you are screwed." The 1-star spike was not gradual — it was triggered by a single design decision that retroactively penalized prior investment. Players who rated 1-star consistently use the phrase "I used to love this game."

**Application for WeaponCraft:** The design line to never cross is: "existing player investment becomes worthless unless they spend more." New weapons should make the meta richer, not make old weapons actively unusable. If a new crafting material appears, it should enable new synergies — not make old recipes obsolete. The F2P trust contract is: new things add options, not remove them.

### Lesson 6: Stamina Systems Work — But Only If the Content They Gate Feels Rewarding

**Evidence:** [GENUINE SOURCE] Reddit: "do dailies without needing to use my laptop. The HSR pattern of 240 Trailblaze Power providing ~6 meaningful farming runs is a useful calibration: players should feel they have "done something" by the time stamina runs out, not feel like they barely started.
