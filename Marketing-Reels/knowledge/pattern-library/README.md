# pattern-library/ — empirical, longevity-weighted patterns

One file per genre (e.g. `tactical-autobattler.md`, `merge.md`, `match3.md`). **Populated by the LEARN phase, appended each run** — empty until the first run.

## Per-genre file format (the LEARN phase writes this)
```
# <genre> — empirical creative patterns
> Source: app-intel ad_top_creatives · network(s): <…> · pulled <date> · N creatives torn down.

## Winning hooks (longevity-ranked)
| pattern | example app | run length (first→last seen) | first-1s description |
|---|---|---|---|
| <hook archetype> | <app> | <e.g. 2025-10 → 2026-06 = 8mo> | <what the opening frame shows> |

## Winning structures
- <structure> — seen in <apps>, typical duration <Ns> — <why it works>

## Saturated / avoid (high frequency, short run = burned out)
- <pattern> — <why it's fatigued>

## Copy / caption phrases that recur
- <phrase> (from <app>)
```

Longer run length = stronger evidence. Always cite the creatives so claims are auditable.
