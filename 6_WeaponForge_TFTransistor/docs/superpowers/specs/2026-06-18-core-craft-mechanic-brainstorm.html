The core loop — mana → fire the mix

The engine that replaces GD's "gears spin → produce troops." No charge-then-explode cascade — the reaction is a component of the fired volley, not a separate pop.

  1. Each hero's weapon auto-does its base attack (slash, shot — the innate delivery). Each base attack fills a mana bar.
  2. When mana is full, the weapon fires its current power-mix — a volley composed from whatever's arranged on its board right now.
  3. The mix = each element tile contributes its element to the volley; modifiers transform components by bridging neighbours.
  4. Mana refills; repeat. Mana fill-rate = the production rate; the arrangement/synergy tunes it.
The worked example you gave 3× Fire in a line → mana full → fires a 3×Fire volley (common, low-mid damage).
Same line, but a modifier bridges one Fire to the Water tile below it → fires 2×Fire + 1×Steam when mana is full. The arrangement composes the volley.

Master list — every decision so far

Everything we've landed across the session, grouped — 57 items across 10 groups. Tags: ✅ agreed · 🟡 leaning · ⏳ open.

A · Architecture & layers

  1. ✅ Three layers: FORGE (build) → LINEUP (assign to lanes) → COMBAT (auto-plays the result).
  2. ✅ Forge is a calm planning layer; combat is fully auto-resolved (no in-combat micro).
  3. ✅ The weapon is the board — you build on the weapon, per hero.
  4. ✅ Hex grid, not square (6 neighbours, one uniform adjacency, TFT-native).
  5. ✅ 3 hero lanes = 3 hex rows (H1 / H2 / H3).
  6. ✅ Hero = the lane carry (gacha collectible, ≈ GD's King); wields the weapon-board.
  7. 🟡 Heroes collected via gacha; runes drafted per-run (TFT shop).

B · Runes (the tiles)

  1. ✅ A rune = Element × Style (two tags), like TFT origin × class.
  2. 🟡 Elements (proto): Fire, Water, Storm, Earth (+ more later).
  3. 🟡 Styles / functions (proto): DOT, Burst, Spread (+ more later).
  4. ✅ Both elements get all styles → e.g. multi-Fire AND multi-Water → 3×3 combination space per element-pair.
  5. ✅ New element/style types revealed slowly via gacha over stages 5+ (anti-overwhelm).
  6. 🟡 Rune visual: fill = element, icon/letter = style, frame = rarity.
  7. 🟡 Runes merge T1→T5 (rarity borders), TFT-style.

C · Modifiers (the connectors)

  1. ✅ Modifier = separate part class; its job is to bridge UNLIKE elements.
  2. ✅ Modifiers have directionality — they connect specific neighbour directions (e.g. top + right).
  3. ✅ Rarity adds directions: rare = 2 dirs, epic = 3 (top-right-bottom), legendary = up to 6 (a hub).
  4. 🟡 Modifier shape = which directions it bridges (the visual tells you).
  5. ✅ Rarity = a frame colour (silver / purple / gold) — never reuse an element colour.
  6. ✅ Same-element adjacent runes link free (no modifier needed).
  7. 🟡 Same-style adjacent runes also give a style synergy (free).

D · Synergy & combos (spatial, never count)

  1. ✅ Synergy is spatial (adjacency) — never positionless "have 3 of X".
  2. ✅ Bridge output: the element pair sets WHAT the combo is (Steam, Conduct, Magma); the style sets the DELIVERY (DoT / Burst / Spread).
  3. ✅ Three pair buckets: Combinable (named combo) · Neutral (generic rider) · Opposite (cancel/fizzle).
  4. ✅ Opposite = a teachable mistake; early levels are designed so the player never hits the bad feeling.
  5. ✅ Forbidden/opposite pairs unlock as premium combos via epic/legendary modifiers (late-game tactic shift).
  6. 🟡 Conduct chains: Storm arcs through adjacent Waters (multi-tile spatial effect).
  7. 🟡 Hub (legendary, 6-ring) = multi-combine; several bridges at once.
  8. ✅ Magicka lives here: the combination table + opposite/conduct rules + the hub = multi-combine.

E · Production / firing (the mana model)

  1. ✅ Base attack fills a mana bar; full mana → fire the current mix.
  2. ✅ The fired mix is composed by the arrangement (3×Fire, or 2×Fire+1×Steam, …).
  3. ✅ Modifiers transform components of the mix (bridged Fire→Water makes that shot Steam).
  4. ✅ No separate reaction meter — the "reaction" is a volley component, not an explosion.
  5. 🟡 Mana fill-rate = production rate; synergy/arrangement tunes how fast you fire.
  6. 🟡 "Mana runes" sell as parts: regen-rate = Modifier (≈ GD speed gear); max-pool = Passive (≈ GD troop cap).
  7. 🟡 Bigger/heavier mix components may cost more to fire (fast-small vs slow-big tradeoff).
  8. ⏳ Stamina/energy = a re-skin of mana for physical heroes (same mechanic).

F · Heroes / weapons / progression

  1. ✅ Per-hero innate weapon-passive = the base attack shape (slash / projectile / AoE) = the delivery + the mana-filler.
  2. 🟡 Hero leveling unlocks more board slots on the weapon (the upgrade incentive).
  3. ✅ Locked slots expand the modifier/output side (not "+1 passive aura").
  4. 🟡 Rune rarity ladder: rare → epic → legendary → (mythic).
  5. ✅ Manual tap-ult survives as a hero ability (charged by play) — a divergence from GD's auto-cast King skill.

G · Lanes / cross-hero

  1. ✅ Cross-hero synergy = a bridge spanning two lanes (vertical hex adjacency).
  2. ✅ Which hero in which lane = the lane-assignment agency (cross-lane optimisation).
  3. 🟡 Chars join over a run → new lane → new seams → re-optimise.
  4. 🟡 Combat stays 3-lane; hex rows map to the 3 lanes (offset bands).

H · Why-rearrange (the engagement engine)

  1. ✅ Piece-churn: new runes/modifiers/heroes keep arriving → re-fit the board.
  2. ✅ Matchup-churn: each wave/stage favours a different element (enemy weak/resist) → re-tune.
  3. ✅ Scarce adjacency: 6 neighbours, synergy goals compete for them → the packing puzzle.
  4. ✅ A visible metric (mana-rate + mix composition → output) is the number you chase.

I · Complexity reveal

  1. ✅ New mechanics drip at stages 5 / 10 / 15 — always a next cool thing to chase.
  2. ✅ Early levels: simple same-element mixes only; no opposite-cancel bad feeling.
  3. 🟡 Reveal order (draft): same-element → modifier bridge → cross-lane → directional/epic modifiers → conduct → hubs → forbidden pairs.

J · Enemies / matchup

  1. ✅ ~5 enemy types, each with element strengths/weaknesses (rock-paper-scissors).
  2. ✅ Enemies have a mega-weakness to a COMBO (weak to Fire, mega-weak to Steam) → craft incentive.
  3. ✅ Enemy composition forces craft adaptation (GD: archers come → you pivot to cavalry).
  4. 🟡 Telegraph the wave's enemies at the forge break so the player can re-tune toward the matchup.

Mixtures — first examples of every type

The "current mix" is what fires on full mana. Examples (numbers are placeholder).

Same-element mixes (the floor)

  • F+F+F → 3×Fire volley — common, low-mid, reliable.
  • W+W → 2×Water — applies Wet stacks (sets up Conduct/Steam later).

Same-style mixes (the other axis)

  • Fire-DOT + Water-DOT adjacent → DoT-style synergy: both tick faster / longer (style match, no modifier needed).
  • Fire-Burst + Storm-Burst → Burst-style synergy: bigger alpha on the full-mana volley.

Bridge — combinable (signature, the Magicka table)

  • Fire + Water (modifier) → Steam. Delivery by style: Fire-DOT+W = Scald (DoT); Fire-Burst+W = Steam-blast (AoE); Fire-Spread+W = Steam-cloud (lingering).
  • Water + Storm (modifier) → Conduct — arcs through adjacent wet tiles (chain).
  • Fire + Earth (modifier) → Magma — heavy sustained burn.

Bridge — neutral (generic floor)

  • Storm + Earth (no signature) → generic combined rider (both statuses, modest). Any pair does something.

Bridge — opposite (cancel; late-game premium)

  • Fire + Frost (common modifier) → fizzle (dead bridge — a mistake to learn).
  • Fire + Frost (epic modifier) → Thermal Shock — a premium late-game combo (forbidden pair forced).

Movements — modifier directionality + re-placing

"Moving a piece" changes the mix. Examples:

  • Directional bridge: a Split-shot, top+right modifier connects only the tiles above and to its right. Put Fire top and Water right → it bridges them → Steam enters the mix.
  • Rotate / re-aim: same modifier rotated so its arms face top+left instead → now it bridges different neighbours → different combo (or none).
  • Rarity upgrade adds a direction: rare (top+right) → epic (top+right+bottom) → now also grabs the tile below → a 3-way combo from the same slot.
  • Slide to cluster: move the hub one cell into the centre of a ring → it goes from bridging 2 to bridging all (the B1→B2 lesson).
  • Form a conduct chain: slide a Storm between two Waters → it arcs to both → a chain that can cross the lane seam.
  • Spread vs concentrate: cluster 3 Fire for a fat 3×Fire volley, OR split them across lanes so each lane fires + bridges its own neighbour.
